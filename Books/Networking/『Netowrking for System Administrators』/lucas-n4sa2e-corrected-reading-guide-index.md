## Reading method

```text
read
→ pause
→ explain with Feynman method
→ connect to one piece of evidence
→ identify the beginner mistake
→ write the short deliverable
```

Do not turn Lucas into another giant lab. Use these as disciplined reading guides.

## Minimum standard

A chapter is finished only when the student can answer:

1. What problem does this chapter help diagnose?
2. What evidence would I collect?
3. What wrong conclusion would a beginner make?
4. How does this apply to `client01`, `app01`, or `db01`?
5. What would I say to a network team if I had to escalate?

Highlighting is not studying. Nodding is not understanding. Running a command without a hypothesis is just typing.
