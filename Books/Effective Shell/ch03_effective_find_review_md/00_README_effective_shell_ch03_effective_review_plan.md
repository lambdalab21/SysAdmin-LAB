The point is not to memorize more examples.

The point is to become effective:

```text
ask a precise question
limit the search scope
choose correct tests
preview results
act safely
verify the result
```

Main bridge from Shotts:

```text
Shotts: learn `find`, `locate`, `xargs`, `-exec`, tests, and actions.
Effective Shell: learn how to shape a search expression and avoid fuzzy thinking.
```

Final standard:

```text
He can explain any `find` command in plain English before running it.
He can predict likely matches and likely misses.
He can preview before acting.
He can diagnose his own bad search expression.
```
