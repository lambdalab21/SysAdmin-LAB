## Method

Each module follows the same rhythm:

```text
read
→ explain with Feynman method
→ slow down and predict
→ observe with Linux commands
→ drill with hypothesis/evidence/conclusion
→ quiz
→ final explanation
```

Do not let the student treat the commands as typing practice. Every command must answer a question.
